Introduction

JAVA is a general-purpose programming language that is object-oriented, class-based, and designed to have as few implementation dependencies as possible.

Java was created at Sun Microsystems, Inc., where James Gosling led a team of researchers in an effort to create a new language that would allow consumer electronic devices to communicate with each other.

Main Ideas behind Java's Object-oriented programming:

OOP concepts include:

1) abstraction
2) encapsulation
3) inheritance and
4) polymorphism.

Major Advantage of using java

Java OOP concepts let us create working methods and variables,
re-use all or part of them without compromising security.
